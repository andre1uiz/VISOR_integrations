#Elite Program for Hand-Eye Calibration with Sensopart VISOR

The objective of this project is to enable in a more automated way the hand-eye calibration for Elite Robotics where the existent plugin could not be added

You can use the task directly or create your own configurations and task. If only the script is downloaded, you must create 8 poses named sequentially as `pose1`, `pose2` ... `pose8` and consider `pose1` as the measurement plane pose.

Units are in meters for this script, but it can be changed to mm addin a few lines

Tested only   in simulation so far.
